# GRC Scanner

Open-source compliance scanner for HIPAA, ISO 27001, and APPs.

## Features
- HIPAA compliance checks
- ISO 27001 controls
- Australian Privacy Principles
- AWS cloud scanning
- YAML/JSON/PDF support

## Quick Start
```bash
mkdir build && cd build
cmake ..
make
./grc-scan --file config.yaml --framework hipaa
```
